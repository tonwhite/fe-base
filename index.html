3 способи використання CSS стилів

  1. Стилі в HTML-тегах

    Приклад: p class="color: red"

  2. Стилі в head файлу HTML

    Приклад: styles p { color: green; }, /styles

  3. Стилі в окремому файлі CSS

    Приклад: link rel="stylesheet" href="./styles/style.css"

    Вказується в head

Селектори CSS стилів

  1. Використання class

    Приклад: p class="text"{}

  2. Використання id

    Приклад: p id="text"{}

  3. Використання data target

    Приклад: p data-target="id"{}

Використання комбінацій селекторів

  1. Комбінація tag.class#id

    Приклад: p.class#id{} - описує тег p, якому присвоєно комбінацію певного class та id

  2. Комбінація tag[type]

    Приклад: button[type="submit"]{} - описує зазначений тег, що містить в собі певний атрибут

  3. Задання стилів одночасно для декількох селекторів

    Приклад: p.text, p#text{} - одночасно описує теги p class="text" та p id="text"

  4. Вкладені теги

    Приклад: div span {} - описує span, який огорнутий в div

  5. Вкладені теги 1 рівня

    Приклад: div > span {} - описує тег, що знаходиться на 1 рівні вкладеності
