3 способи використання CSS стилів

  1. Лінійний CSS

    Приклад: p class="color: red"

  2. Внутрішній CSS

    Приклад: styles p {color: green;} /styles

  3. Зовнішній CSS

    Приклад: link rel="stylesheet" href="./styles/style.css" Вказується в head

Типи селекторів

  1. Використання назви елементу

    Приклад: p{} - описує всі теги p

  2. Використання class

    Приклад: .class{}

  3. Використання id

    Приклад: #text{}

  4. Використання data target

    Приклад: data-target="id"{}

  5. Використання вкладеності

    Приклад: p[class="text"]{};

Використання комбінацій селекторів

  1. Комбінація tag.class#id

    Приклад: p.class#id{} - описує тег p, якому присвоєно комбінацію певного class та id

  2. Комбінація tag[type]

    Приклад: button[type="submit"]{} - описує зазначений тег, що містить в собі певний атрибут

  3. Опис одночасно для декількох селекторів tag.class, tag#id

    Приклад: p.text, p#text{} - одночасно описує теги p class="text" та p id="text"

  4. Селектор вкладених тегів tag tag

    Приклад: div span {} - описує всі span, які знаходяться в div

  5. Селектор вкладеного тегу 1 рівня tag tag

    Приклад: div (знак більше) span {} - описує тег span, що знаходиться на 1 рівні вкладеності відносно div

  6. Селектор тегу, що йде після певного елементу tag.class + tag#id

    Приклад: p.text + p#text - описує тег p, що має id="text", та йде після тегу p, що має class="text"

  7. Селектор тегу, чий class починається з певного слова tag[class^="word"]

    Приклад: p[class^="some"]{} - описує всі теги p, яким присвоєні class, що починаються зі слова "some"

  8. Селектор тегу, що містить певне слово в назві class tag[class*="word"]

    Приклад: p[class*="some"]{} - описує всі теги p, яким присвоєні class, що починаються зі слова "some"

  9. Всеохоплюючий першопроходець або селектор "Зірик"

    Приклад: *{} - тег, що описує стилі для всіх сторінок (саме з нього починають всі поважаючі себе розробники)
